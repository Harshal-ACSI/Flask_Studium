What is the Periodic Table?

The Periodic Table, also known as the Periodic Table of Elements, is a chart that shows all the known elements. More specifically, The Periodic Table is a tabular array of the chemical elements organized by atomic number, from the element with the lowest atomic number, hydrogen, to the element with the highest atomic number, oganesson. [1] To learn more about the Periodic Table, we need to know what Elements exactly are.

What is an Element?

An element is defined as a pure substance composed of atoms that all have the same number of protons in the atomic nucleus. [2] The Periodic Table has a total of 188 elements (This number might change in some Periodic Tables depending on when the Periodic Table was created. Currently, in 2024, there are only 118 elements). Ninety-eight of these elements are found naturally on Earth. Of these 98, 88 are found in measurable amount while the rest are found in trace amounts. The final 28 elements are all synthetic. Now that we know what an Element is, let's see the Periodic Table.

The Periodic Table

Let's zoom into a single tile of the Periodic Table. Say hello to Carbon.
A tile on the Periodic Table is made of mainly four parts, these are the Atomic Number, Element Symbol, Element Name and Atomic Mass (or Atomic Weight)*.

Now, Let's look at slightly larger building blocks of the Periodic Table.

Periods, Groups & Series

The Elements in the Periodic Table can be grouped in many ways. They can either be grouped as periods, groups or series. Let's see what each of them means.

Periods

Simply put, periods are the "rows" on a periodic table. The elements Carbon(C), Nitrogen(N), and Oxygen(O) are from the same period in the periodic table. The atoms in the same period have the same number of electron shells. For example, the elements Carbon, Nitrogen and Oxygen all have 2 electron shells as they are in Period 2 (second row of the periodic table).

Groups

Groups are the "columns" of the periodic table. Carbon(C), Silicon(Si), and Germanium(Ge) all are from the same group. Elements in the same group all have the same number of valence electrons. This makes the elements in the same group react similarly. For example, the elements Carbon, Silicon and Germanium all have 4 valence electrons as they are in Group 14 (14th column of the periodic table).

Footnotes

* There is a key difference between Atomic Mass and Atomic Weight (Atomic Weight is also known as Relative Atomic Mass).
Definition of Atomic Mass: Atomic mass is the mass of an atom or an isotope which includes the total mass of the nucleus and the electrons present in the periphery.
Definition of Atomic Weight: Atomic weight, also called relative atomic mass, is the weighted arithmetic mean of the relative isotopic mass of all the isotopes of a particular element depending on the abundance of each of those isotopes. To learn more, read this article.
