What is the Periodic Table?

The Periodic Table, also known as the Periodic Table of Elements, is a chart that shows all the known elements. More specifically, The Periodic Table is a tabular array of the chemical elements organized by atomic number, from the element with the lowest atomic number, hydrogen, to the element with the highest atomic number, oganesson. [1] To learn more about the Periodic Table, we need to know what Elements exactly are.

What is an Element?

An element is defined as a pure substance composed of atoms that all have the same number of protons in the atomic nucleus. [2] The Periodic Table has a total of 188 elements (This number might change in some Periodic Tables depending on when the Periodic Table was created. Currently, in 2024, there are only 118 elements). Ninety-eight of these elements are found naturally on Earth. Of these 98, 88 are found in measurable amount while the rest are found in trace amounts. The final 28 elements are all synthetic. Now that we know what an Element is, let's see the Periodic Table.
