The periodic table is a chart that shows all the known elements.

An Element

An element is defined as a pure substance composed of atoms that all have the same number of protons in the atomic nucleus. [1] The periodic table has a total of 188 elements (This number might change in some periodic tables depending on when the periodic table was created. Currently, in 2024, there are only 118 elements). 98 of these elements are found naturally on Earth. Of these 98, 88 are found in measurable amount while the rest are found in trace amounts. The final 28 elements are all synthetic.
